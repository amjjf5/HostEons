**使用建议和注意事项**

买了VPS之后，有几个点需要注意一下。

首先是备份问题。HostEons提供免费的快照功能，但这个快照是存在同一个数据中心的。如果机房出现物理故障，快照也会受影响。建议定期把重要数据备份到其他地方，比如用rclone同步到云存储服务。

其次是安全设置。VPS到手之后，第一件事就是改掉默认的SSH端口，禁用root密码登录，改用密钥认证。再装个fail2ban防止暴力破解，配个基础的防火墙规则。虽然有100Gbps的DDoS防护，但自己的安全措施也不能少。

然后是资源监控。特别是Budget系列，因为是Fair Share模式，最好装个监控工具看看资源使用情况。如果发现CPU经常跑满，或者磁盘IO很高，可能就需要考虑升级套餐了。

最后是关于Windows系统。Ryzen系列支持安装Windows Server，但需要选择年付、双年付或三年付的账单周期。提交订单后发个工单，客服会帮你激活免费的Windows 2019 Server Standard授权。注意这个Windows授权只有年付以上才免费，月付是不提供的。
